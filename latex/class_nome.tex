\doxysubsection{Detailed Description}
Representa o domínio de \doxylink{class_nome}{Nome} no sistema. 

Esta classe armazena e valida uma string de nome. Regras de validação\+:
\begin{DoxyEnumerate}
\item Tamanho\+: 5 a 20 caracteres.
\item Caracteres permitidos\+: Letras (A-\/Z, a-\/z) e espaço em branco.
\item Formatação\+:
\begin{DoxyItemize}
\item O primeiro caractere de cada termo deve ser letra maiúscula (A-\/Z).
\item Espaços em branco devem ser seguidos por uma letra (não podem ser seguidos por outro espaço ou estar no fim).
\item O último caractere não pode ser espaço. 
\end{DoxyItemize}
\end{DoxyEnumerate}
