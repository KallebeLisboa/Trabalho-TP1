\doxysubsection{Detailed Description}
Representa o domínio de \doxylink{class_telefone}{Telefone}. 

Esta classe armazena e valida uma string representando um número de telefone. Regras de validação\+:
\begin{DoxyEnumerate}
\item Formato\+: "{}+\+DDDDDDDDDDDDDD"{} (sinal de mais seguido por 14 dígitos).
\item Total de 15 caracteres.
\item Cada caractere após o \textquotesingle{}+\textquotesingle{} deve ser um dígito (0-\/9). 
\end{DoxyEnumerate}
