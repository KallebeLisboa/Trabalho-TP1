\doxysubsection{Detailed Description}
Representa o domínio de Código identificador no sistema. 

Esta classe armazena e valida uma string que representa um código único. Regras de formato\+:
\begin{DoxyEnumerate}
\item Deve conter exatamente 10 caracteres.
\item Cada caractere deve ser uma letra (a-\/z) ou um dígito (0-\/9). 
\end{DoxyEnumerate}
