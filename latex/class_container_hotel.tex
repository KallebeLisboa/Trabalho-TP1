\doxysubsection{Detailed Description}
Classe responsável pelo armazenamento e gestão em memória de objetos \doxylink{class_hotel}{Hotel}. 

Esta classe implementa o padrão Container, atuando como um banco de dados em memória (usando std\+::map). Ela é responsável por gerenciar o ciclo de vida (CRUD) das instâncias da entidade \doxylink{class_hotel}{Hotel}.
