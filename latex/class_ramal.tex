\doxysubsection{Detailed Description}
Representa o domínio de \doxylink{class_ramal}{Ramal} telefônico. 

Esta classe armazena e valida uma string numérica que representa um ramal. Regras de validação\+:
\begin{DoxyEnumerate}
\item O valor deve representar um número inteiro.
\item O intervalo válido é de 00 a 50. 
\end{DoxyEnumerate}
