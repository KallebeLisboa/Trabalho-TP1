\doxysubsection{Detailed Description}
Classe responsável pelo armazenamento e gestão em memória de objetos \doxylink{class_quarto}{Quarto}. 

Esta classe implementa o padrão Container para a entidade \doxylink{class_quarto}{Quarto}. Diferente de outras entidades, o \doxylink{class_quarto}{Quarto} não possui uma chave única global (vários hotéis podem ter o quarto "{}101"{}). Por isso, este container implementa uma lógica de chave composta (Código do \doxylink{class_hotel}{Hotel} + Número do \doxylink{class_quarto}{Quarto}) para garantir a unicidade no mapa.
