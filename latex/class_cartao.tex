\doxysubsection{Detailed Description}
Representa o domínio de número de Cartão de Crédito. 

Esta classe é responsável por armazenar e validar números de cartão de crédito. As regras de validação aplicadas são\+:
\begin{DoxyEnumerate}
\item O número deve conter exatamente 16 dígitos numéricos.
\item O número deve ser válido segundo o algoritmo de Luhn (dígito verificador). 
\end{DoxyEnumerate}
