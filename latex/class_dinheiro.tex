\doxysubsection{Detailed Description}
Representa o domínio de \doxylink{class_dinheiro}{Dinheiro} (valor monetário). 

Esta classe armazena valores monetários. Para evitar erros de arredondamento típicos de ponto flutuante, o valor é armazenado internamente como um inteiro representando centavos. O intervalo válido é de 0,00 a 1.\+000.\+000,00.
