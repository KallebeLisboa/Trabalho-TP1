\doxysubsection{Detailed Description}
Entidade base que representa uma \doxylink{class_pessoa}{Pessoa} no sistema. 

Esta é a classe mãe (base) para as entidades \doxylink{class_gerente}{Gerente} e \doxylink{class_hospede}{Hospede}. Ela encapsula os dados comuns a todos os usuários do sistema\+: \doxylink{class_nome}{Nome} e \doxylink{class_email}{Email}. O atributo \doxylink{class_email}{Email} atua como chave primária (PK) para identificar unicamente uma pessoa.
