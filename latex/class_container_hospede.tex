\doxysubsection{Detailed Description}
Classe responsável pelo armazenamento e gestão em memória de objetos \doxylink{class_hospede}{Hospede}. 

Esta classe implementa o padrão Container, atuando como um banco de dados em memória (usando std\+::map). Ela é responsável por criar, recuperar, atualizar, e remover instâncias da entidade \doxylink{class_hospede}{Hospede}.
