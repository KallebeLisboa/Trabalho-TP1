\doxysubsection{Detailed Description}
Classe responsável pelo armazenamento e gestão em memória de objetos \doxylink{class_reserva}{Reserva}. 

Esta classe implementa o padrão Container para a entidade \doxylink{class_reserva}{Reserva}. Atua como um banco de dados em memória, permitindo criar, listar, atualizar e excluir reservas, mantendo a integridade dos dados e as referências para Hóspedes e Quartos.
