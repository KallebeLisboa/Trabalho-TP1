\doxysubsection{Detailed Description}
Representa o domínio de \doxylink{class_senha}{Senha} de usuário. 

Esta classe armazena e valida uma string de senha. Regras complexas de validação definidas no projeto\+:
\begin{DoxyEnumerate}
\item Tamanho\+: Exatamente 5 caracteres.
\item Caracteres permitidos\+: Letras (A-\/Z, a-\/z), dígitos (0-\/9) e caracteres especiais (!, @, \#, \$, \%, \&).
\item Requisito de composição\+: Deve conter pelo menos uma letra maiúscula, uma minúscula, um dígito e um caractere especial.
\item Regra de não-\/repetição de tipo\+: Letra não pode ser seguida de letra, e dígito não pode ser seguido de dígito. 
\end{DoxyEnumerate}
