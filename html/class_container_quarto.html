Detailed Description

Classe responsável pelo armazenamento e gestão em memória de objetos Quarto.

Esta classe implementa o padrão Container para a entidade Quarto. Diferente de outras entidades, o Quarto não possui uma chave única global (vários hotéis podem ter o quarto "101"). Por isso, este container implementa uma lógica de chave composta (Código do Hotel + Número do Quarto) para garantir a unicidade no mapa.
