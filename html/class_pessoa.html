Detailed Description

Entidade base que representa uma Pessoa no sistema.

Esta é a classe mãe (base) para as entidades Gerente e Hospede. Ela encapsula os dados comuns a todos os usuários do sistema: Nome e Email. O atributo Email atua como chave primária (PK) para identificar unicamente uma pessoa.
