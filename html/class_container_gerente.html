Detailed Description

Classe responsável pelo armazenamento e gestão em memória de objetos Gerente.

Esta classe implementa o padrão Container, atuando como um banco de dados em memória (usando std::map). Ela é responsável por criar, recuperar, atualizar, remover e autenticar instâncias da entidade Gerente.
