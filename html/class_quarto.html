Detailed Description

Entidade que representa um Quarto no sistema.

A classe Quarto agrupa as informações de uma unidade de hospedagem. Ela contém dados sobre o número do quarto, capacidade de hóspedes, valor da diária e ramal telefônico. Além disso, armazena o código do Hotel (Chave Estrangeira) para vincular este quarto a um estabelecimento específico.
