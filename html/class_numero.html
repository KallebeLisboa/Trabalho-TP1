Detailed Description

Representa o domínio de Número (geralmente associado a um quarto).

Esta classe armazena e valida uma string numérica que representa um número. Regras de validação:

  1. O valor deve representar um número inteiro.
  2. O intervalo válido é de 001 a 999.
