<h3>Trabalho Técnicas de Programação </h3>

 O sistema de software visa facilitar a gestão de pequenos hotéis. Para usar o sistema, o gerente deve  criar uma conta. Após
 criar uma conta, deve ser autenticado informando o seu email e a sua senha. Após ser autenticado, o gerente pode acessar os
 serviços disponibilizados pelo sistema. A seguir, tem-se uma relação desses serviços: ler, editar e excluir conta; criar, ler, editar
 e excluir hotel; criar, ler, editar e excluir quarto; criar, ler, editar e excluir reserva; criar, ler, editar e excluir hóspede; listar
 hotéis (apresenta dados de cada hotel); listar quartos (apresenta dados de cada quarto); listar reservas (apresenta dados de cada
 reserva); listar hóspedes (apresenta dados de cada hóspede). Não podem ocorrer conflitos entre reservas. Não é possível editar
 dado que identifica registro (chave primária). Finalmente, o sistema deve assegurar as multiplicidades informadas nos
 relacionamentos presentes no seguinte diagrama de classes e também não deve possibilitar exclusões que resultem em
 inconsistências
